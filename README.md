## Overview

The Ordinals Explorer is a Next.js (app router) project that uses the [Hiro Ordinals API](https://docs.hiro.so/ordinals).

There are two main parts of this project:

- The explorer `app/(explorer)` ‚Äî A frontend for exploring ordinal inscriptions
- The preview API `app/(preview)` ‚Äî An endpoint to render a specific ordinal inscription
